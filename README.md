# Calc-Ult

---

The Calculator Ultimate (calc ult) aims to unleash the power of an average beginner project in a fun way

---

## About

I noticed the informatic problem is so easy compared to the potential it can achieve.
Therefore because this is a terminal user interface application i decided to revolve my first project around it
So that i can learn the following: strings, bits opperands, manuanlly coded math functions, vector handling

---

Compared to other project this project currently supports complex int operations, basic real number arithmetic, big number addition (up to 10 to the 3rd digits)
Areas to extend: Geometry, complex plane, vector calc, TUI tweaks, more documentation, financial mathematics mode (plan for v1.2.0) and complex bits handling

---

## How do I run?

-> Download
-> compile again the cpp with g++ in terminal for insurance "g++ calc-ult-1filer.cpp -o calc"
-> run with ./calc [mode]
Modes: masscli (vector mode), mastercli (double mode), help, static (a, b, operation), interactive cli (int mode)
